The State of Michigan is a required collaborator for the [[initiatives/update-building-codes|Update Building Codes]] initiative because Michigan municipalities are constrained by the State's building code and cannot independently adopt more stringent local standards ([[sources/cap/cap-2020|cap-2020]]). Ann Arbor's A2Zero plan calls for working with the State to adopt the 2021 building code and zero code appendix, with the State expected to adopt the updated code in 2021 and provide educational materials and inspector training ([[sources/cap/cap-2020|cap-2020]]).
