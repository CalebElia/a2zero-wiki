# Historical Implementation Plans

This directory contains implementation plans and specs from earlier development
sessions, kept for historical reference. **They do not reflect current code or
schema.** Many use the retired `bronze/`/`silver/`/`gold/` medallion vocabulary
and reference functions/files that have since been renamed.

For current project state, see:
- [/SCHEMA.md](../../SCHEMA.md) — page types, frontmatter, ontology governance
- [/CLAUDE.md](../../CLAUDE.md) — project brief loaded into every Claude session
- [/CHANGELOG.md](../../CHANGELOG.md) — what changed when, and why

These plans are preserved because they document the reasoning behind major design
decisions and may help future contributors understand why the codebase looks the
way it does. Read them as archaeology, not specification.
