Synthesis Generation Modes (GraphRAG Parallel)
Ingest-Triggered (Community Summaries)
Auto-generated when entity thresholds are crossed during ingest. Bootstraps the synthesis layer without waiting for user queries.
  • After all 7 strategy pages updated → "A2Zero Strategy Framework Overview"
  • When 3+ funding events share fund-type → "Federal Grant Strategy in A2Zero"
  • When initiative spans 4+ years → "Evolution of [Initiative]"
  • When 3+ commitments are missed/carried-forward → "Persistent Implementation Challenges"
  • After each council meeting ingested → "Key Decisions: [Meeting Date]"
  • After each actor page created → "Role of [Actor] in A2Zero"
Query-Triggered (User Queries)
Generated when a user question doesn't match an existing current synthesis. Saved as draft, requires human review before promotion.
  • "Tell me about all geothermal efforts" → comparative synthesis
  • "Why did the SEU ballot pass?" → mechanism synthesis
  • "What are all the federal grants Ann Arbor has won?" → funding synthesis
  • "Where has Ann Arbor stalled?" → commitment-failure synthesis
  • Low-quality answer → dark matter dossier entry (wiki gap signal)
