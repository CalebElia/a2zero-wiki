New Page Type: location — Geographic Hierarchy

Location pages anchor initiatives, funding events, and actors to specific places. The parent-location field creates the geographic hierarchy — from Washtenaw County down to a specific building. Multiple initiatives that share a location reveal co-location synergies the wiki would otherwise miss.

Synthesis Freshness — The Post-Ingest Pipeline

This pipeline runs automatically after every ingest run — no human involvement until the review queue at the end. Every step is a script, not an LLM call. The LLM only runs when new synthesis needs to be generated.

① LLM Ingest Runs
New source document (council transcript, press release, annual report) processed against SCHEMA.md rules. Wiki pages created or updated. Raw quads emitted as a batch.
new/updated wiki pages new quads batch (raw)
② Deduplication Script
Normalizes entity names using the alias table (SEU = Sustainable Energy Utility). Groups new quads by (subject, relation, object) after normalization. Merges duplicates — appends source_uuid to existing quad's sources array. Recalculates confidence = count of independent sources. Appends net-new quads to quads.jsonl.
quads.jsonl updated alias table updated confidence scores recalculated
③ Staleness Detection Script
For each newly added or updated quad: extracts entities touched (subject, object, tags). Scans all wiki/synthesis/*.md files for invalidation-triggers that match. Flips matching synthesis pages from status: current → status: needs-review.
Example: New quad ingested: {2025-03, Bryant-geothermal, federal-funding, frozen, source: news-2025-03-12}
Script finds synthesis page "Ann Arbor Geothermal Pursuits" has invalidation-triggers: [entity: district-geothermal, tag: geothermal]
→ Status flipped to needs-review. Next retrieval shows warning banner: "Updated sources available — this summary may be outdated."
N synthesis pages → needs-review
④ Dark Matter Regeneration
Full gap detection runs on the updated quads.jsonl. Finds pairs of causally-connected events with unexplained temporal distance. LLM call: reasons about what domain logic says should be between each gap pair. Regenerates blackboard/dark_matter.md completely — gaps collapsed by new quads are removed; new finer-grained gaps from this ingest run are added.
dark_matter.md regenerated collapsed gaps removed new gaps added
⑤ Ingest-Triggered Synthesis Check
Checks whether any community-summary thresholds were crossed in this ingest run:
• New initiative page created spanning 4+ years? → Draft "Evolution of [Initiative]"
• 3rd funding event of same fund-type? → Draft "[Fund-Type] Strategy in A2Zero"
• New meeting page created? → Draft "Key Decisions: [Date]"
• New actor page created? → Draft "Role of [Actor] in A2Zero"
• 3rd commitment marked missed/carried-forward → Draft "Persistent Challenges"
If threshold met: LLM generates draft synthesis, saves as status: draft.
N new draft synthesis pages
⑥ Review Queue Updated — Human Picks Up Here
needs-review (3)→ Regenerate with new context, or approve as-is if changes are minor
new drafts (2)→ Review LLM-generated ingest synthesis, promote to current or edit
dark matter (5 gaps)→ Act on dossier: search suggested sources or schedule interview

Steps ①–⑤ are fully automated scripts. No human involvement until step ⑥. LLM calls only in ①, ④ (gap reasoning), and ⑤ (draft synthesis generation).
